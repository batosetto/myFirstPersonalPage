CPAN 111 Database Concepts and Design The study of database design and management of a database is an essential component of the business IT world today. Through this course, the student will gain a strong theoretical background in database design. The student will work with Entity-Relationship Diagrams (ERD) to learn and implement the basic database design. Students will also use the Normalization principles to enhance and further develop the database design. The concepts of keys and integrity constraints will be investigated. ERDs will be drawn to model the relational database. SQL
CPAN 112 Fundamentals of Numeric Computing This course covers the fundamental mathematics needed for programmers who plan to work in a business environment. The course begins with a review of arithmetic and linear algebra. Percentages and proportions are then introduced to solve problems such as price changes and currency conversions. Systems of linear equations are graphed and solved to make the most efficient use of resources. With these mathematical tools covered, the course moves on to business applications such as break-even charts, factors involved in setting a selling price and calculating the interest on loans and investments. The concepts in this course will be reinforced through Python programming exercises that highlight the importance of programmers in the business world. Python
CPAN 116 Algorithms and Problem Solving The purpose of this course is to introduce students to problem solving with structured techniques and algorithm development using pseudocode and graphical tools. Discussion will center on tools and techniques used in the designing and modelling of business applications. This is an introductory course, which is meant to teach the techniques of solving problems using tools like flowcharts, task lists and pseudocode. Students will also be introduced to Object Oriented concepts. Assignments and/or case studies will be used to support the application of theory. Java
CPAN 133 Operating Systems Operating Systems is a course designed to give the user knowledge of the most used and powerful operating systems in existence today. In this course the student will acquire an understanding of UNIX. The student will learn how to create shell script programs, and how to control and manipulate the user environment, including file system, security threading and scheduling. Linux
CPAN 134 Web Programming and Design This course is an introduction to web design. Students study technical aspects of web design and interface development, with a focus on valid markup and programming in HTML, CSS and JavaScript. Beginning with basic HTML and CSS,students learn valid mark-up while focusing on layout and effective site navigation. Students will develop an understanding of JavaScript along with an introduction to multimedia content HTML, CSS and JavaScript. Portfolio building projects are emphasized. Version control systems are introduced. HTML and JavaScript
ESOL 120 Technical Reading and Writing Skills This course is designed to develop the writing skills that will be required for clear communication in technical documents. Students will learn to write documents that are coherent, accurate, and grammatically correct. -
